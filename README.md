# Movie App React Nativc

> Simple Movie App built on React Native + UI Kitten + Redux

## How to use it?

```bash
# 1. install dependencies
$ npm install

# 2. create .env file from .env.example, fill with your own TMDB API KEY

# 3. serve the app
$ npm start
```
